#  Sistema de Gestión de Gastos Full Stack

Este es un proyecto de gestión de finanzas personales que permite registrar, visualizar y eliminar gastos en tiempo real. Desarrollado para demostrar habilidades en el stack **Node.js** y **Vanilla JavaScript**.

##  Funcionalidades
- **Registro de gastos:** Permite ingresar concepto, precio y cantidad.
- **Cálculo automático:** Suma el total acumulado de todos los registros.
- **Persistencia de datos:** Los datos se guardan en un archivo JSON en el servidor.
- **Interfaz Moderna:** Diseño responsive creado con CSS3 puro.

##  Tecnologías
- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **Backend:** Node.js, Express.js.
- **Gestión de Versiones:** Git y GitHub.

##  Instalación
1. Clona el proyecto.
2. Ve a la carpeta `backend` y ejecuta `npm install`.
3. Inicia el servidor con `node server.js`.
4. Abre `frontend/index.html` en tu navegador.
