#  Sistema de Gestión de Gastos Full Stack

Este proyecto es una aplicación robusta para el control de finanzas personales. Permite a los usuarios registrar conceptos de gastos, precios y cantidades, calculando automáticamente el total acumulado. Está diseñado con una arquitectura profesional que separa el Frontend del Backend.

>  **Repositorio del código:** [https://github.com/FacundoConsultas/control-gastos-fullstack.git](https://github.com/FacundoConsultas/control-gastos-fullstack.git)

##  Características principales
- **Gestión Integral (CRUD):** Registro y eliminación de gastos en tiempo real.
- **Cálculo Automático:** El sistema procesa los subtotales y el total general al instante.
- **Arquitectura Profesional:** Separación clara entre cliente (Frontend) y servidor (Backend).
- **Persistencia de Datos:** Uso de archivos JSON para el almacenamiento de información.
- **Diseño Responsive:** Interfaz moderna y adaptable a dispositivos móviles.

##  Tecnologías utilizadas
- **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript Vanilla (ES6+).
- **Backend:** Node.js, Express.js, CORS.
- **Persistencia:** File System (fs) de Node.js.
- **Control de Versiones:** Git & GitHub.
